Insertion sort

Insertion sort is a simple sorting algorithm that builds the final sorted array (or list) one item at a time. It is much less efficient on large lists than more advanced algorithms

    Select the first Unsorted Element Swap other elements to right create the correct position and shift unsorted element Advance the marker to the right one element

Selection sort

Selection sort is a simple sorting algorithm. This sorting algorithm is an in-place comparison-based algorithm in which the list is divided into two parts, the sorted part at the left end and the unsorted part at the right end

    Select the Lowest element in the remaining array Bring it to the starting position Change the counter for unsorted array by one

Quick Sort

Like Merge Sort, QuickSort is a Divide and Conquer algorithm. It picks an element as pivot and partitions the given array around the picked pivot. There are many different versions of quickSort that pick pivot in different ways.

    Bring the pivot to its appropriate position such that left of the pivot is smaller and right is greater Quick Sort the left part Quick Sort the right part

Bubble Sort

Bubble Sort is the simplest sorting algorithm that works by repeatedly swapping the adjacent elements if they are in the wrong order. Called bubble sort because it acts as though the larger numbers bubble to the top.

    Compares adjacent elements. Swaps if they are out of order Repeat over through array until all is sorted.

Merge Sort

Merge sort is a divide-and-conquer algorithm based on the idea of breaking down a list into several sub-lists.

    Divides input array in two halves, calls itself for the two halves until array size is 1 and then merges the two sorted halves.
